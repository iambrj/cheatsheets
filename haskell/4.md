# Infix notation
```
> 4 + 3 == 7
True
```
and
```
> (==) (4 + 3) 7
```
are equivalent

# Guards
Guards are a way of testing whether some property of a value (or several of
them) are true or false
```
mod x
	| x < 0 = 0 - x
	| otherwise = x  -- Syntactic sugar for true
```

# `where` clause
```
numOfRealSolutions a b c
	| disc > 0 = 2
	| disc == 0 = 1
	| disc < 0 = 0
		where
		disc = b ^ 2 - 4 * a * c
```
